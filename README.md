# AI Background Remover

An AI-powered web application that removes image backgrounds instantly using the Remove.bg API.

## Live Demo

🌐 https://ai-background-remover-roan.vercel.app/

## Features

* AI-powered background removal
* Google Authentication
* Image upload and processing
* Download processed images
* Modern responsive UI
* Secure API integration

## Tech Stack

* Next.js
* TypeScript
* Firebase Authentication
* Remove.bg API
* Tailwind CSS
* Vercel

## Installation

Clone the repository:

```bash
git clone https://github.com/vivekpatel24165/ai-background-remover.git
cd ai-background-remover
```

Install dependencies:

```bash
npm install
```

Create a `.env.local` file:

```env
REMOVE_BG_API_KEY=your_api_key
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
